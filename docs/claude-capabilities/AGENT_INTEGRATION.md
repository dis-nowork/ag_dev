## Best Practices

### 1. Minimal Context Loading

```python
# âŒ Bad: Load everything
skills = load_all_skills()

# âœ… Good: Load on demand
skill = load_skill_if_needed(user_intent)
```

### 2. Clean Output Paths

```python
# âŒ Bad: Random temp files
output = "/tmp/abc123xyz.png"

# âœ… Good: Organized structure
output = "/output/project_name/image_001.png"
```

### 3. Cost Tracking

```python
# Track costs per operation
cost_tracker.add("imagen", 0.02)
cost_tracker.add("kling_video_5s", 0.45)
cost_tracker.add("elevenlabs_100chars", 0.01)

# Report at end
print(f"Total cost: ${cost_tracker.total()}")
```

### 4. Graceful Degradation

```python
# Always have a fallback
try:
    result = primary_api()
except:
    try:
        result = secondary_api()
    except:
        result = placeholder_with_warning()
```
