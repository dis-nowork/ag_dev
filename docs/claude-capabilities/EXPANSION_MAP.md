| `copy/objection` | Matar objeções | FAQ copy, Preemptive strike |

### 1.2 Frameworks Implementados

| Framework | Uso | Origem |
|-----------|-----|--------|
| AIDA | Attention-Interest-Desire-Action | Classic |
| PAS | Problem-Agitate-Solve | DR Standard |
| 4U | Urgent-Unique-Useful-Ultra-specific | Headlines |
| QUEST | Qualify-Understand-Educate-Stimulate-Transition | Long form |
| BAB | Before-After-Bridge | Transformation |
| FAB | Features-Advantages-Benefits | Product |
| SLAP | Stop-Look-Act-Purchase | Ads |
| PASTOR | Problem-Amplify-Story-Testimony-Offer-Response | Complete |
| Star-Chain-Hook | Star story-Chain benefits-Hook CTA | Gary Halbert |
| Greased Slide | Every line sells the next | Joe Sugarman |

### 1.3 Estilos Black/Gray

| Estilo | Uso | Cuidados |
|--------|-----|----------|
| Clickbait-Deliver | Hook agressivo, entrega valor | Evitar banned words |
| Fear-Based | Medo de perder | Dosagem certa |
| Greed-Based | Desejo de ganhar | Prova social |
| Curiosity Loop | Aberto que força clique | Fechar no conteúdo |
| False Scarcity | Urgência fabricada | Platform TOS |
| Social Proof Stack | Depoimentos em massa | Verificáveis |
| Authority Hijack | Associar a experts | Não mentir |
